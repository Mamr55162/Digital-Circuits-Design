# Logic-Digital-Circuits-Design

## 🚀 Project Title
A C++ implementation of combinational and sequential logic circuits.

## What does it consist of?
This project focuses mainly on implementing ICs both combinational & sequential, starting from simple logic gates
moving to the more complex operations built on logic gates, then finally implementing a real-life IC that applies the logic as needed.

## 🛠️ What was used in this project?
* [C++/JetBrains CLion]

## 📌 Features
- All logic gates are built using a universal gate **NAND**.
- ICs are implemented to mimic real-life circuits as accurately as possible.
- Each IC & circuit has a simulation designed in Multisim.
- Contains real-life Indicators such as LEDs, probes, Bus monitors, etc.
- Contains a guide to the multisim designs, with a document for each circuit implementation & design.

## 💻 How to run it?
until now, the only way to try it is through a terminal by compiling and running the .cpp file, I am currently still working on a GUI for the project.
